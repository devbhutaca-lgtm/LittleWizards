## How to Use the Flyers

### Option 1: View in Browser
1. Open the HTML files directly in any web browser
2. Files are located in `/app/flyers/`
3. View and review the design

### Option 2: Save as PDF (Recommended)
1. Open the HTML file in Chrome or Edge browser
2. Press `Ctrl+P` (Windows) or `Cmd+P` (Mac)
3. Select "Save as PDF" as the printer
4. Adjust settings:
   - Paper size: Letter (8.5 x 11 inches)
   - Margins: None
   - Background graphics: Enabled
5. Click "Save"
6. Your PDF flyer is ready to print or share!

### Option 3: Print Directly
1. Open HTML file in browser
2. Press `Ctrl+P` (Windows) or `Cmd+P` (Mac)
3. Select your printer
4. Recommended settings:
   - Paper: Letter size (8.5 x 11")
   - Quality: High/Best
   - Color: Color printing
5. Print!
